Starting our react project

Step 1: make sure to have Node installed
https://nodejs.org/en/

Step 2:
Install repo from github
https://github.com/AntiEbay/AntiEbay

Step 3: Open the front-end folder in Terminal/IDE with terminal option

***NO Quotes***
Step 4: run "npm install" 

Step 5: run "npm install -D tailwindcss postcss autoprefixer"

Step 6: run "npm start"

The page should start running on localhost:3000

Step 7: run "npm stop" This will kill the program.
